У меня есть несколько игр для Windows, купленных в Epic Games Store, которые я хочу запускать через Steam, чтобы использовать его как единый лаунчер для всех игр.

## Почему через Steam

У меня компьютер подключен к телевизору, и я играю в игры в основном в режиме Big Picture от Steam. Этот режим удобен для навигации с помощью геймпада и позволяет легко переключаться между играми.

Мышка и клавитура все равно лежат не очень далеко, если захочу что-то быстро настроить, но их нужно доставать и подключать. В основном я использую геймпад.

## Почему просто не добавить ярлык в библиотеку

Ярлыки для запуска игр из Epic Games на рабочем столе выглядят как ссылки c протоколом `com.epicgames.launcher://`, например:

```
com.epicgames.launcher://apps/b0cd07...50951?action=launch&silent=true
```

Такие ссылки не работают в Steam, если просто добавить их как ярлык для запуска. Steam просто не дает добавить такое приложение.

## Решение

Чтобы запускать такие игры, нужно добавить в библиотеку Steam лаунчер Epic Games, а в параметры запуска этого ярлыка добавить ссылку на нужную игру.

Для начала добавьте ярлык с игрой на рабочий стол из Epic Games.

### Добавляем приложение в библиотеку Steam

Открывайте Steam и выбирите в меню `Games` → `Add a Non-Steam Game to My Library`.

В открывшемся окне выберите `Browse` и найдите путь к исполняемому файлу Epic Games Launcher. Обычно он находится по адресу:

```
C:\Program Files (x86)\Epic Games\Launcher\Portal\Binaries\Win64\EpicGamesLauncher.exe
```

Теперь в Steam появится ярлык для запуска Epic Games Launcher, но чтобы сразу запускать нужную игру, нужно добавить в параметры запуска ссылку на игру.

### Настраиваем ярлык для запуска игры

Щелкните правой кнопкой мыши на этом приложении в библиотеке Steam и выберите `Properties`.

В поле `Launch Options` добавьте ссылку на игру из ярлыка для запуска. А в начале этой строки добавьте дефис. Например:

```
-com.epicgames.launcher://apps/b0cd07...50951?action=launch&silent=true
```

Теперь при запуске этого приложения из Steam будет запускаться Epic Games Launcher с нужной игрой.

### Обложки и логотипы игры в Steam

В окне `Properties` в разделе `Shortcut` укажите имя игры. А рядом с именем нажмите на иконку, чтобы установить логотип игры.

Чтобы найти логотип игры, можно воспользоваться сайтом [SteamGridDB](https://www.steamgriddb.com/). Найдите там нужную игру и скачайте понравившийся логотип и другие изображения, которые понадобятся позже.

В разделе `Customization` установите следующие изображения:

- `Cover` (grid, 600x900) — изображение для отображения игры на полке.
- `Background` (hero, 3840x1240) — фон на странице с информацией об игре.
- `Logo` (logo, 1280x720) — изображение полного логотипа игры, которое будет отображаться в верхней части страницы игры.
- `Wide Cover` (grid, 920x430) — широкая обложка для игры, например, для оборажения в списке недавних игр в режиме Big Picture.

Теперь вы можете запускать игры из Epic Games прямо из Steam и наслаждаться удобным интерфейсом для навигации и запуска игр.
