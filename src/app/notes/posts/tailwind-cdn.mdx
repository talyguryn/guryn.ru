[Tailwind CSS](https://tailwindcss.com/) — это популярный CSS-фреймворк, который позволяет быстро создавать адаптивные и стилизованные интерфейсы.

Чтобы подключить Tailwind CSS к вашей HTML-странице без использования сборки, добавьте в секцию head следующий код:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<script src="https://unpkg.com/@tailwindcss/browser@4"></script>
```

После этого вы можете использовать классы Tailwind CSS в вашем HTML-коде.

## Пример использования

```html
<html>
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <script src="https://unpkg.com/@tailwindcss/browser@4"></script>
  </head>
  <body>
    <div
      class="bg-blue-500 text-white px-6 m-4 py-4 h-36 rounded-lg flex flex-col justify-end items-start"
    >
      <span class="text-3xl font-bold">Привет, Tailwind CSS!</span>
      <span class="">Это пример использования Tailwind CSS без сборки.</span>
    </div>
  </body>
</html>
```
