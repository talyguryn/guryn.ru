Для хранения файлов и фильмов дома, я решил использовать отдельный компьютер с Ubuntu. В нем есть быстрый диск и гигабитный ethernet-порт для подключения напрямую к роутеру.

Чтобы другие устройства в локальной сети могли получить доступ к определенной папке с файлами, я настроил Samba сервер.

## Установка Samba

Обновил список пакетов и установил сервис.

```
apt update
apt install -y samba
```

Открыл файл с настройками Samba.

```
nano /etc/samba/smb.conf
```

И добавил настройки папки, которой буду делиться. Такие разделы с настройками можно добавлять на каждую папку, которую нужно расшарить. `Shared folder` — это имя папки, которое будет видно в сети. А `/shared` — это путь к папке на сервере.

```
[Shared folder]
  path = /shared
  browseable = yes
  read only = no
  guest ok = yes
  create mask = 0777
  directory mask = 0777
  force user = root
```

Создал папку, которой буду поделиться.

```
mkdir /shared
chmod 777 /shared
```

Перезагрузил сервис.

```
systemctl restart smbd
```

Теперь эта папка доступна в локальной сети с других компьютеров и телефонов.
